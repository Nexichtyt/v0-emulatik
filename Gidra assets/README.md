# Gidra assets

Все графические ассеты, используемые в проекте. Структура папок повторяет
расположение в `public/`, чтобы их можно было положить туда без изменений путей в коде.

## icons/  →  public/icons/
Иконки приложений на рабочем столе и в доке.

| Файл             | Где используется                                  |
|------------------|---------------------------------------------------|
| google-play.png  | Иконка «Google Play» (рабочий стол + папка)       |
| standoff2.png    | Иконка «Standoff 2» + превью в окне приложения    |
| settings.png     | Иконка «Настройки»                                |
| files.png        | Иконка «Файлы»                                    |

## previews/  →  public/previews/
Превью-изображения внутри окон приложений.

| Файл             | Где используется                                  |
|------------------|---------------------------------------------------|
| google-play.png  | Превью контента вкладки                            |

## logos/  →  public/
Брендовые логотипы.

| Файл              | Где используется                                 |
|-------------------|--------------------------------------------------|
| logo-s-crown.png  | Логотип на экране загрузки/брендинг              |
| logo-s-elite.png  | Альтернативный логотип бренда                    |

## favicons/  →  public/
Иконки сайта (favicon / PWA), подключены в `app/layout.tsx`.

| Файл                  | Назначение                                  |
|-----------------------|---------------------------------------------|
| icon.svg              | Основная векторная иконка                    |
| icon-light-32x32.png  | Favicon для светлой темы                     |
| icon-dark-32x32.png   | Favicon для тёмной темы                      |
| apple-icon.png        | Apple touch icon                             |

> Примечание: стандартные плейсхолдеры Next.js (`placeholder.svg`,
> `placeholder.jpg`, `placeholder-user.jpg`, `placeholder-logo.*`) не включены,
> так как это служебные заглушки фреймворка, а не реальные ассеты проекта.
